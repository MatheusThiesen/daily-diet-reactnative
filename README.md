# ⚙️ Como rodar

### 📃 Pré-requisito

- **Node** v18.16.1
- **Expo Go App**

### 💻 No PC

- **Clonar repositório**
- **Dentro da pasta do projeto execute:**

```bash
$ npm install
```

- **E então, execute:**

```bash
$ npx expo start
```

### 📱 No Smartphone

- **Download aplicativo _Expo Go_ no seu smartphone** </br>
- **No Expo Go, click em _Scan QR code_, e escaneie o que está aparecendo no terminal do seu PC** </br>
- **Pronto!**
